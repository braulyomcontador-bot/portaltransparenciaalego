Como o portal funciona

O portal se atualiza sozinho — ele lê os dados abertos do Governo de Goiás direto no seu navegador. Você não precisa instalar nada.

1
Abra o arquivo. Ao abrir, ele tenta conectar à API de dados abertos e baixar as emendas mais recentes. O selo no topo mostra o estado: ao vivo, demonstração ou erro.
2
Se conectar (ao vivo): as emendas reais aparecem automaticamente. Clique em Atualizar quando quiser puxar de novo.
3
Se não conectar: alguns portais bloqueiam acesso direto pelo navegador. Nesse caso, baixe o arquivo .csv em transparencia.go.gov.br ou no EmendasGO e use o botão Importar — o portal entende as colunas sozinho.

Despesas e perfil

A API de dados abertos só traz as emendas. As despesas (verba indenizatória, diárias, folha de gabinete) e o perfil de cada deputado saem da Transparência da ALEGO em CSV — basta importar pelo mesmo botão. Até lá, esses campos aparecem como “sem dados”.

Publicar na internet

É um arquivo único. Para deixar no ar, suba em qualquer hospedagem estática gratuita (GitHub Pages, Netlify, Cloudflare Pages) — é só arrastar o arquivo.
